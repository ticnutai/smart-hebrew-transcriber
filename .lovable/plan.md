

## תוכנית: שדרוג ניהול תמלולים מתקדם

### מצב קיים
- FolderManager בסיסי: תיקיות כטקסט חופשי, תגיות ידניות, חיפוש פשוט
- אין מועדפים, אין קטגוריות קבועות, אין עריכת כותרת/הערות inline, אין מיון

### מה ייבנה

#### 1. קטגוריות קבועות (לצד תיקיות)
- רשימת קטגוריות מוגדרות מראש: ישיבה, הרצאה, ראיון, שיחה, אחר
- אפשרות להוסיף קטגוריות מותאמות אישית
- סינון לפי קטגוריה בנוסף לתיקיות

#### 2. מועדפים (Favorites)
- כוכבית ליד כל תמלול לסימון כמועדף
- טאב/פילטר "מועדפים" מהיר

#### 3. ניהול תגיות משופר
- הצעות תגיות אוטומטיות (מתוך תגיות קיימות)
- סינון לפי תגית ספציפית בלחיצה על תגית
- תגיות צבעוניות

#### 4. עריכה inline
- עריכת כותרת ישירות מהרשימה (לחיצה כפולה)
- הוספת/עריכת הערות לכל תמלול
- שינוי קטגוריה מתוך dropdown

#### 5. מיון ותצוגה
- מיון לפי: תאריך, שם, אורך טקסט, מנוע
- תצוגת רשימה / תצוגת כרטיסים (grid)
- בחירה מרובה למחיקה/העברה של כמה תמלולים

### שינויים טכניים

| קובץ | שינוי |
|---|---|
| **מיגרציה** | הוספת עמודות `category` ו-`is_favorite` לטבלת `transcripts` |
| `src/hooks/useCloudTranscripts.ts` | עדכון טיפוסים + תמיכה בשדות החדשים |
| `src/components/FolderManager.tsx` | שכתוב מלא - הוספת קטגוריות, מועדפים, מיון, תצוגות, עריכה inline, בחירה מרובה |
| `src/pages/Dashboard.tsx` | התאמת ה-props החדשים |

### שינוי DB
```sql
ALTER TABLE transcripts 
  ADD COLUMN category text DEFAULT '',
  ADD COLUMN is_favorite boolean DEFAULT false;
```
